* 쿼리 메소드 기능
    - 메소드 이름으로 쿼리 생성
    - 메소드 이름으로 JPA NamedQuery 호출
    - @Query 어노테이션을 사용해서 리파지토리 인터페이스에 쿼리 직접 정의

    - find...By
    - read...By
    - query...By
    - get...By

    - count...By -> long
    - exists...By -> boolean
    - delete...By, remove...By -> boolean
    - findDistinct, findMemberDistinctBy
    - findFirst3, findFirst, findTop, findTop3
